## 리액트에서 자주 사용하는 JS문법

### 구조 분해 할당

구조 분해 할당은 배열이나 객체를 분해하여 그 값을 개별 변수에 할당하는 것을 말한다.
흔히 이런 코드를 접해보았을 것이다.

```ts
const { a, b } = { a: 1, b: 2 };
```

객체 뿐만 아니라 배열도 구조 분해 할당을 사용할 수 있는데, 값이 없는 경우 기본값을 설정해줄 수도 있다.

```ts
const [a = 1, b = 2, c = 3] = [1, 2];
console.log(a, b, c); // 1 2 3
```

여기서 주의해야할 점은 기본값은 변수에 값이 할당되지 않은 상태. 즉, undefined인 경우에만 적용된다는 점이다.

객체 구조 분해 할당에서도 마찬가지로 기본값을 설정할 수 있으며, 이를 새로운 변수에 할당하는 것도 가능하다.

```ts
const { a: first, b = 2, c = 3 } = { a: 1, b: 2 };
console.log(first, b, c); // 1 2 3
```

이 객체 구조 분해 할당은 리액트에서도 매우 자주 사용되는 문법으로 기억해두자.

```tsx
function Component({
  a = 1,
  b = 2,
  c = 3,
}: {
  a: number;
  b: number;
  c: number;
}) {
  return (
    <div>
      {a} {b} {c}
    </div>
  );
}
```

### 전개 연산자

흔히 Spread Operator라고 불리는 전개 연산자는 배열이나 객체를 펼치는 역할을 한다.

```ts
const arr = [1, 2, 3];
const newArr = [...arr, 4, 5, 6];
console.log(newArr); // [1, 2, 3, 4, 5, 6]
```

스프레드 연산자 이전에 배열을 합치기 위해선 concat, splice 등 복잡한 과정이 필요했으나, 전개 연산자를 사용하면 간단하게 합칠 수 있다.

반대로 배열을 분해하여 객체에 할당할 수도 있다.

```ts
const [first, ...rest] = [1, 2, 3, 4, 5];
console.log(first, rest); // 1 [2, 3, 4, 5]
```
